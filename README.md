# Cine Stream Next

A movie browsing application built with Next.js using TMDB APIs.

## Features

- Browse popular movies
- Search movies
- View movie details
- Favorite movies
- Filter by genre
- Filter by release year
- Responsive UI
- Redux Toolkit state management
- LocalStorage persistence
- Jest & React Testing Library support

---

## Tech Stack

- Next.js 15
- React 19
- Redux Toolkit
- React Redux
- TMDB API
- Jest
- React Testing Library

---

## Installation

```bash
npm install
```

Create:

```
.env.local
```

Add:

```
TMDB_KEY=YOUR_API_KEY
```

Run:

```bash
npm run dev
```

---

## Testing

Run tests

```bash
npm test
```

Generate coverage

```bash
npm test -- --coverage
```
